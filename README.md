# Trabalho Estrutura de dados 2

O objetivo deste trabalho é juntar vários arquivos que já estão em ordem em um único arquivo final, também ordenado. Para isso, o programa compara os registros desses arquivos e escolhe, a cada passo, o menor código de cliente, gravando-o no novo arquivo. Esse processo continua até que todos os registros tenham sido copiados. A ideia é mostrar uma forma organizada e eficiente de unir várias partes de um mesmo conjunto de dados, criando um resultado único e completo no final.
